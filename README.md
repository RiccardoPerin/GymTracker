# GymTracker 🏋️

A comprehensive, offline-ready workout tracking application built with Flutter for iOS, designed to manage high-frequency weightlifting routines (like a consistent 4-day split). It allows users to create structured templates and track live sessions with an integrated rest timer.

## 🚀 Features

*   **Routine Templates:** Create and save base workouts (e.g., Push, Pull, Legs) to quickly start a session.
*   **Live Session Tracking:** Log sets, reps, and weights dynamically during the workout.
*   **Rest Timer:** Built-in floating countdown timer for tracking rest periods between sets.
*   **Cloud Sync & Storage:** Powered by Supabase to securely back up training data and analyze progress over time.
*   **Clean UI/UX:** Intuitive interface specifically designed for ease of use while at the gym.

## 🛠 Tech Stack

*   **Frontend:** Flutter & Dart
*   **Backend / Database:** Supabase (PostgreSQL)
*   **State Management:** Provider
